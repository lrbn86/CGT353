What is web development?

Web development is the process of creating a website or a web application.

There are many approaches to web development. However, any website or web application will always consist of the three following core technologies:

  • HTML (Hypertext Markup Language)
    • Provides structure to the web page
  • CSS (Cascading Style Sheet)
    • Provides presentation to the web page
  • JS (JavaScript)
    • Provides interactivity to the web page

A website can be of two types:

  • Static - Involves only HTML, CSS, and JS
    • An example of a static site is this one.
  • Dynamic - Involves both client-side and server-side technologies
    • An example of a dynamic website would Youtube.

The architecture of a website or a web application can be divided into two layers of web development:

  • Frontend Development
    • This is what the user sees. It can involve web design as well, but that can be a separate layer on its own.
    • Developed using client-side technologies such as HTML, CSS, and JS
  • Backend Development
    • The user cannot see this directly as the frontend abstracts it. The user usually interacts with the backend when they submit a form or buy a product online.
    • Developed using back-end technologies such as PHP, NodeJS, and Java.
    • A database such as MySQL, MongoDB, or Firebase may be used to support the backend of a website or web application.

One can specialize in one specific web development layer or specialize in both layers, which is often called full-stack development.

There exists many web frameworks and libraries such as React, Vue, Angular, Flask, Django, and jQuery that can help speedup development by providing common structures (i.e. not developing from scratch).
